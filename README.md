# ArenaIQ 🏟️

AI-powered smart stadium system to prevent crowd disasters.

## Features
- Real-time crowd monitoring
- AI congestion prediction
- Smart exit routing using Google Maps
- Live alerts system

## Impact
Prevents overcrowding and improves safety during large events.

## Tech Stack
- Node.js
- Express
- Google Maps API
- Firebase

## How to Run
npm install
npm start
